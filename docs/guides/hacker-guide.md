> 如何注册 Profile、报名活动、组队和提交项目。

## 注册 Profile

```bash
./scripts/create-profile.sh <github-username>
```

Profile Schema 详见 [profiles/_schema.yml](../profiles/_schema.yml) 和 [PRD 6.2](specs/synnovator-prd.md)。

## 报名活动

通过 GitHub Issue 提交报名，使用 [Registration 模板](../.github/ISSUE_TEMPLATE/register.yml)。

## 提交项目

```bash
./scripts/submit-project.sh <hackathon-slug> <team-name>
```

提交规范详见 [PRD 6.3](specs/synnovator-prd.md)。
